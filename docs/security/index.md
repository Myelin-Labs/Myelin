This section is the honest answer to *"what is Myelin actually
proving?"* It's deliberately precise: every claim Myelin makes has
a corresponding evidence path, and every evidence path has a
boundary where it stops being a claim.

## Why this section exists

Most protocol docs bury the security boundary in marketing copy.
Myelin's project-level docs (`MYELIN_USE_CASE_POSITIONING.md`,
`MYELIN_PRODUCTION_GATE.md`, etc.) make a point of saying exactly
what the current evidence is and isn't. This section is the
user-facing version of the same discipline.

Read this section **before** you trust any number in the rest of
the docs.
