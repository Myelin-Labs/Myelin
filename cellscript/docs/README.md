# CellScript Documentation Map

This directory keeps the active CellScript references used by Myelin's typed
Cell execution path. Historical release notes, archived plans, standalone
website material, and proposal-era roadmaps have been removed from this tree.

## Stable Tutorials

`docs/wiki/` contains reader-facing tutorials and cookbook material for the
current compiler surface.

## References

Top-level `docs/CELLSCRIPT_*.md` files describe active compiler behaviour,
target-profile evidence, runtime errors, syntax governance, metadata, capacity,
deployment manifests, and support matrices.
